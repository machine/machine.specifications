Code Snippet Files for Visual Studio (tested in VS2010).Created from the ReSharp LiveTemplates from http://therightstuff.de/CategoryView,category,MSpec.asp

Use and modify create_mspec_install_bat.py (defaults to VS2010) if you need to.
After executing the script an install_vs[Version]_snippets.bat file is created in from the content of the mspec folder.Execute the install_vs[Version]_snippets.bat in order to copy the snippets to your USERPROFILE location of Visual Studio
