MbUnit Samples
==============

This directory contains links to MbUnit sample code projects.

How to use:

1. Download and unzip the desired sample to a directory of your choice.

2. Open the sample solution in Visual Studio.  Some samples may
   include solutions and project files for multiple versions of
   Visual Studio.  Be sure to open the appropriate solution for
   the version of Visual Studio that you wish to use.

3. Build the sample in Visual Studio.

4. Examine and/or edit the code of the sample.  Documentation is
   included within each test fixture and component.

4. Run the sample using Icarus, TestDriven.Net, ReSharper 
   or any other Gallio test runner.
